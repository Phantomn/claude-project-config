# Panel Synthesis: 진행 선언 후 미실행 방지 규칙 보강
Date: 2026-03-12
Agents: CoT, ToT, Step-Back, ReAct, Self-Ask, OODA (총 6명; 스레드 제한으로 Ulysses 제외)
Target: design/pentest/common.md

## 수렴점 (6/6 에이전트 동의)
- **선언(다음 단계/다음 분석 방향)과 실제 실행 사이의 강제 규칙이 부재**하다.
- 해결책은 **선언 → 즉시 실행 → 증빙 기록 → 전환**을 강제하는 구조가 핵심이다.

## 다수 합의 (≥4/6 동의)
- “다음 분석 방향”에 **Action ID**를 부여하고, 다음 응답에서 **Done/Blocked 상태 + 증거/사유**를 의무화한다.
- 미실행 시 **사유 + 대체 행동**을 즉시 기록하고, 반복 미실행은 Anti-Loop에 연결한다.
